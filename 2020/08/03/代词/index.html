代词

2020-08-03

人称代词

  • I We You He She It They

    注意: 在完全倒装句中用正常语序
    Here you are

    反身代词

  • ~ self

    指示代词

  • this / these “这个”

    打电话中的自我介绍
    This is xxx speaking

  • that / those ‘’那个”

    在比较句型中用that和those来代替前面相同名词

    不定代词(某人 某物 某事)

  • some / any

    some用于肯定句
    any用于否定句/一般疑问句 + bady/thing时就是不定代词

  • it / one

    it 指 同类同物
    one 指 同类不同物

  • 可数 && 不可数

有一点 没有 no
不可数 a little little
可数 a few few
* 二者 && 三者
之一 都 都不
二者 either both neither
三者以上 any all none
对话 none = = no one

有of就用none none of

  • other
other adj. 其他的 + 可数名词 other peoples = others 其他人 other things 其他事
the other n. 两者中的另一个
another n.三者或者三者以上另一个 another + 数字 = 数字 +more
others n.其他都不是
the others n. 其他都是

除了other是adj.其他都是代名词
people 是单复数同形 so some people.
another 买衣服时告诉服务员从一堆衣服里面拿一个 another one 再来一个
another + 数字 = 数字 +more another two = two more再来两个

  • it
1 上文提到过的事物 同类同物
2 形式主语/形宾(后面必须有真正的主语 to do /that)
3 It is … that 固定搭配
4 情景对话固定 词组 : make it 成功做某事 get it 理解明白 take it 带走
* 练习题
