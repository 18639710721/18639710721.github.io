魔法方法介绍

  • doc 用来查看类的说明文档
  • module 用来查看当前操作的类所在的模块
    一个.py文件就是一个模块
  • 当执行模块是类所在模块时，执行结果为main.否则，执行结果是类所在模块的名称*
  • class查看当前对象的类
    类也是对象，Demo是type类的对象，d是Demo类的对象
  • dict 用来获取类或者实例的属性，返回值为字典
    对象.dict获取的是普通字段, 只有普通字段存储在对象中，除普通字段以为的成员都存储在类中，所有通过类.dict来获取
  • del()方法，析构方法。当由类创建的实例对象，被删除或者在内存中被释放，将会自动触发执行
    当代码全部执行完毕才自动触发del() ，如果需要提前触发，则需要通过del关键字，删除所有对象后触发del()
    • call()方法用于将对象变成一个可调用的对象，也就是说，当一个类中call()方法时，其实例化得到的对象便是可调用的(callable)
      带()是调用 不带是引用，@property
    • new()方法用于创建与返回一个对象。在类准备将自身实例化时调用

1.new_()和init()的区别
new__方法，它会构造一个对象并返回，init 接收构造的对象并对其进行初始化。
2.init__ 与 new 的关系总结：作用方面，new负责构造对象，init 负责初始化对象；执行顺序方面，new先执行，init 后执行；返回值方面，new返回构造的对象，init 只能返回 None；
3.init是实例方法，new是静态方法
4.返回对象自动触发init()方法，init()方法当中的self，实际上就是new返回的实例，就是该对象。

  • 承一些不可变的class时(比如int, str, tuple)，new方法提供了自定义这些类的实例化过程的途径。
    单例模式
    • 该类只有一个实例且该实例易于外界访问
    • 应用：控制实例个数并
    • 如何用代码实现单例模式呢？
      对象不存在时，–> 去创建对象
      对象存在时–>返回已经创建的对象
